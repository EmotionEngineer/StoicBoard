# Privacy Policy for StoicBoard

## 1. No Data Collection and Transmission
StoicBoard operates entirely offline. The application does not declare, request, or utilize the Android Internet permission (`android.permission.INTERNET`). 

Because there is no network access capability, it is technically impossible for the application to collect, store, transmit, or share your keystrokes, personal details, typed text, or usage analytics with any external servers, developers, or third parties.

## 2. Device Permissions Used
The application requests only the minimum set of permissions required to function locally:
* **VIBRATE:** Used to provide tactile haptic feedback during key presses.
* **QUERY_ALL_PACKAGES:** Used strictly on-device to determine the package name of the active foreground application. This is necessary to apply the user-defined delay profiles (e.g., applying longer delays for messaging applications and no delays for writing tools). This lookup is done in real-time, and the package names are never stored or logged.

## 3. Third-Party SDKs and Ads
The application does not integrate with any third-party advertising, telemetry, analytics, or tracking software.

## 4. Children's Privacy
StoicBoard does not handle, collect, or share personal information from any user, including children under the age of 13.

## 5. Contact
If you have any questions regarding this offline policy, please open an issue in the official GitHub repository.
